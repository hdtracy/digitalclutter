# Helm
[Helm documentation](https://helm.sh/docs)

- **Helm** is a solution to install complex applications on Kubernetes. 
	- **Helm charts** are packages to install on Kubernetes.
		- artifacthub.io (like DockerHub, but for helm charts). 
- Helm charts require an additional binary to install complex applications
	- kubernetes-dashboard install from artifacthub.io (first install [[Helm]]) for practice
